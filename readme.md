# Hi, I'm Jakub 👋

Backend software engineer based in Poznań, Poland. 12+ years building
distributed, high-traffic systems — mostly in Java on GCP.

## What I'm doing now

- 🔍 **Search Platform Engineer at IKEA** (via TantusData) — part of the team
  behind global product search serving millions of users across 90+ markets.
  Search engine in Java on GCP, with Dataflow + Airflow
  pipelines, Terraform IaC, and AI/ML for query intelligence and relevance.
  Multiple production releases per day, decisions driven by A/B tests.
- 🛠️ **Building [g9s](https://github.com/breku/g9s)** — a terminal UI for
  Google Cloud (Cloud Run, Cloud Build, Compute Engine, Secret Manager).
  Browse, log, edit and redeploy resources without leaving the terminal.
- 🚗 **Maintaining [Take&Drive](https://takeanddrive.eu)** — a mobility app
  I co-founded that aggregates shared vehicles and public transit on a single
  map. 100k+ downloads. Kubernetes-based microservices, real-time multi-provider
  data aggregation, map-based APIs.
